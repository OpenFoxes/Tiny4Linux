## What's Changed

This release introduces an entirely new way to use and integrate Tiny4Linux into everyday workflows: Starting with version 2.0, all webcam control functions are available not only through the graphical interface but also as a CLI tool.

### Breaking Changes

The OSCServer got entirely removed. Please refer to the original repository https://github.com/cgevans/tiny2 in order to use that feature.

- #36: Removed no longer supported parts of the project by @Hirschfuchs in https://github.com/OpenFoxes/Tiny4Linux/pull/45

### New Features

The big change with version 2.0 is the CLI Tool. Please refer to the README in order to install it and lookup the description at https://github.com/OpenFoxes/Tiny4Linux/blob/22f24be22eb327ee7e4bbed5531bc7429d128065/docs/cli.md

- #34: Implemented a CLI to control the camera by @Hirschfuchs in https://github.com/OpenFoxes/Tiny4Linux/pull/48
- New parts in Readme by @Hirschfuchs in https://github.com/OpenFoxes/Tiny4Linux/pull/51
