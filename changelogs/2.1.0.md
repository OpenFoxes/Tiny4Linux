## What's Changed

This release focuses on the rework of the GUI. With more features, a more intuitive and flexible UI became necessary.

### UI Version 2

The new UI is available in two modes — Dashboard Mode, containing all the available features, and Widget Mode, targeted for the smallest possible window for everyday use.
